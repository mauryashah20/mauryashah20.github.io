# 📂 Featured Projects

## Maurya Shah

### Smart BLE Garden Valve

Off-grid smart irrigation controller using **ESP32 BLE communication** enabling mobile-based watering schedule control without WiFi.

### ESP-NOW RC Link

Low latency peer-to-peer remote control protocol using **ESP32 ESP-NOW communication** for real-time device control.

### Sign Language Detector

Real-time computer vision system translating hand gestures into text using **Python, OpenCV, and MediaPipe**.

### LAN Chat System

Local area network chat application enabling instant messaging and communication between devices.

### ESP32 BLE Gamepad

Custom hardware gamepad built with **ESP32 Bluetooth Low Energy (BLE)** for ultra-low latency inputs.

### Gimbal Stabilizer

Active camera stabilization system using **IMU sensors and PID control loops**.

### Joystick Control Interface

Hardware interface system translating analog joystick inputs into digital commands for remote systems.

### AI Resume Analyzer

AI-powered resume analysis system using **OCR and NLP** to evaluate resumes against job descriptions and generate ATS scoring.

### Footstep Power Generation

Embedded energy harvesting system converting mechanical footsteps into electrical power using **piezoelectric sensors and Arduino**.

### Space Explorer

Interactive simulation/game designed to explore celestial bodies and astrophysics simulations.
